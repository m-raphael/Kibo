# CyberPet

Open-source macOS companion for Kibo.

## Goal
Use local face tracking and non-sensitive visual cues to generate and animate a robotic mascot with calm companion behaviors.

## Safety rules
- No race or ethnicity inference
- No sensitive biometric categorization
- Local-first by default
- No raw frame storage unless explicitly enabled
