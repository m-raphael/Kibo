# DESIGN.md

## Visual direction
Use an Apple-inspired design language:
- clean, restrained, premium
- soft depth
- subtle blur and translucency
- strong typography hierarchy
- minimal chrome
- calm motion
- generous spacing

## UI rules
- Prefer quiet surfaces over loud gradients
- Use elevation sparingly
- Keep controls compact and precise
- Use system-like spacing and rhythm
- Favor clarity over novelty

## Mascot presentation
- mascot is the main focal point
- background stays quiet
- animation should feel smooth and intentional
- expression changes should be subtle
- avoid cartoon noise

## Color
- base on neutral surfaces
- use a single accent color
- support dark mode first

## Motion
- idle breathing
- gentle head turns
- small expression transitions
- no excessive bounce
