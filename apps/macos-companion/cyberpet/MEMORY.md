# MEMORY.md

## Product identity
CyberPet is a macOS companion app for Kibo.

## Stable decisions
- Repo docs are the source of truth.
- Backlog CSV is the task queue.
- DESIGN.md is the visual source of truth for UI.
- Plan first, then implement.
- Keep tasks PR-sized.
- Use review and approval before continuing.
- Security scan before commit, security review before merge.

## Non-goals
- Do not let PDFs be the only source of truth.
- Do not use external notes as the only project memory.
- Do not expand scope without approval.
